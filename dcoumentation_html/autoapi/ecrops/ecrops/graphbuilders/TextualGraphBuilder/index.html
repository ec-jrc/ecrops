TextualGraphBuilder

class ecrops.ecrops.graphbuilders.TextualGraphBuilder.TextualGraphBuilder(output_file=None)¶

Bases: ecrops.graphbuilders.AbstractGraphBuilder.AbstractGraphBuilder

Example implementation of AbstractGraphBuilder. This builder writes in textual from the structure of the steps and their connections. In the constructor the user should provide the path to the output file. If output file is None (default behaviour) the builder prints the text to the console
